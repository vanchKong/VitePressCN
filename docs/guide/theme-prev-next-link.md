
# 上（下）一篇 {#prev-next-link}

你可以自定义上（下）一篇链接的文本。如果 你想在 上（下）一篇 链接上显示与侧边栏上不同的文本(默认显示侧边栏的文本)，这将很有帮助。

## prev

- 类型： `string`

- 更多：

  指定要在指向上一篇的链接上显示的文本。

  如果你没有在 frontmatter 中设置它，文本默认显示为侧边栏的文本。

- 例：

```yaml
---
prev: 'Get Started | Markdown'
---
```

## next

- 类型： `string`

- 更多：

  与 `prev` 相同，但用于下一篇。

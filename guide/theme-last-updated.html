最近更新时间 #

最近一条内容的更新时间会显示在页面右下角。要启用它，请将 lastUpdated 选项添加到您的配置中。

全局配置 #

将 lastUpdated 选项添加到您的配置中。

js
export default {
  lastUpdated: true
}

Frontmatter 配置 #

如果您想隐藏上次更新的文本，请将 lastUpdated 选项设置为 false。

yaml
---
lastUpdated: false
---

  "title": "最近更新时间",
